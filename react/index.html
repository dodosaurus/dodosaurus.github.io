Chlebiatko Blog

Little blogging platform for hobby baker (my wife). Web app was made from zero without CMS. Frontend was created with React and Bulma, state management with Redux and Thunk. Firebase and its modules are used as backend layer (authentication handling, DB, storage and in this case also hosting). Auth. model is used under the hood (not visible to end user), so only admin can access it and add/edit/delete posts. Test coverage was developed with Cypress.
